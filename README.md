# Laravel + Breeze + ReactJS Starter Template

This is a starter template for building modern web applications using **Laravel**, **Breeze**, and **ReactJS**. It is intended for students to quickly get started with a full-stack Laravel application that includes authentication and a React frontend.

---

## Features

- Laravel 10 backend
- Breeze authentication scaffolding
- ReactJS frontend integrated with Laravel
- API routes ready for React consumption
- Starter components for building your UI

---

## Prerequisites

Before you begin, make sure you have the following installed:

- PHP >= 8.1
- Composer
- Node.js >= 18
- npm or Yarn
- MySQL / PostgreSQL / SQLite
